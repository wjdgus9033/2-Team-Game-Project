게임 구조 및 설명

게임 설명

  • 게임 시작 시 우주선 등장
  • 우주선은 좌우 방향키로 이동 가능
  • 스페이스바로 총알 발사
  • 외계인 또는 보스를 죽이면 점수를 얻는다.
  • 외계인 또는 보스 총알에 충돌 시 게임 종료

조작 방법

  • 키보드 : ← → 좌우 방향키로 이동
  • 스페이스바 : 총알 발사

  • 우주선 (플레이어)
